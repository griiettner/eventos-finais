# Eventos Finais Project Overview

Eventos Finais is a Progressive Web Application (PWA) for interactive Bible study focused on end-times events. Built with React 19, TypeScript, and Vite, it features offline-first architecture with SQLite WASM for local data storage, Kinde OAuth authentication, and modern glassmorphism UI with Framer Motion animations.

## Key Features
- Offline-first PWA with SQLite WASM database
- Kinde OAuth authentication (Google, Facebook, Apple SSO)
- 20 chapters of study content with audio playback
- Interactive questions and progress tracking
- Admin dashboard for content management
- Modern glassmorphism UI design
- Portuguese language support

## Technology Stack
- **Frontend**: React 19, TypeScript, Vite
- **Database**: SQLite WASM with Web Worker
- **Authentication**: Kinde OAuth
- **UI**: CSS-in-JS with glassmorphism, Framer Motion
- **Icons**: Lucide React
- **PWA**: Vite PWA plugin
- **State Management**: React Context API

## Project Structure
- `src/pages/` - Main application pages (LandingPage, Dashboard, ChapterDetail, AdminDashboard)
- `src/contexts/` - Authentication and state management
- `src/db/` - SQLite database service and worker
- `src/services/` - Business logic (AdminService)
- `src/components/` - Reusable UI components
- `src/data/` - Static data and interfaces
- `src/hooks/` - Custom React hooks

## Database Schema
- `chapters` - Study chapters with content and audio
- `questions` - Chapter-specific study questions
- `user_answers` - User responses to questions
- `progress` - Reading and audio progress tracking
- `user_profile` - User information and admin status

## Authentication Flow
- Uses Kinde OAuth with social providers
- Supports Apple, Google, Facebook sign-in
- Admin access controlled by Kinde roles
- Custom authentication pages for seamless UX

## Admin Features
- Chapter management (create, edit, delete)
- Audio file upload and management
- Question management system
- Role-based access control via Kinde

## Development Notes
- Offline-first architecture with sync capabilities
- Database operations run in Web Worker for performance
- Responsive design works on all devices
- Progressive Web App with install capability
