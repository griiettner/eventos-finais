# Project Structure & Key Components

Project directory structure: src/components/ (SyncManager), src/contexts/ (AuthContext, AuthContextCore), src/data/ (chapters), src/db/ (db-service, db-worker), src/hooks/ (useAuth), src/pages/ (LandingPage, LoginPage, Dashboard, ChapterDetail). Key files: App.tsx with protected routes, main.tsx entry point, and global styles. Database schema includes chapters, user_answers, progress, and user_profile tables.
