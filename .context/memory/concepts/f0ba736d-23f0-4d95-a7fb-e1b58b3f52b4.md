# Project Overview & Technical Stack

The 'Eventos Finais' project is a Bible study web application focused on the book 'Eventos Finais'. 
Tech Stack:
- Frontend: React, Vite, Framer Motion, TailwindCSS.
- Backend/Storage: Firebase, Firestore (using a custom server-firestore.cjs), SQLite (WASM).
- PWA: Configured with vite-plugin-pwa.
- Icons: Standard 192x192, 512x512, Maskable 192x192/512x512, and Apple Touch Icon 180x180.
- Theme: Dark (#1a1a1a).
- Environment: Docker support with Dockerfile and docker-compose.
